PWA en vue3

  • Crear un proyecto nuevo
  • Generar el proyecto manualmente
  • incluir PWA
  • Todo lo demás por defecto
  • dirigir se al directorio de trabajo si no se a limpiado GIT
  • ejecutar el constructor con: npm run build
  • instalar el servidor de prueba : npm install http-server -g
  • Borrar el caché de chrome para evitar conflictos (ver store/windows/borrar datos de caché)
  • levantar el dist con http-server dist/
  • el servicio se levanta para iniciar la configuración del manifest.json y Service Worker
  • Se debe a que un service worker solo funciona a través de un https, un navegador seguro

    • TUNELADO

  • Se hace con ngrok: https://download.ngrok.com/windows
  • Descargar el ejecutable e instalarlo
  • Mostrará una consola donde ejecutaremos: ngrok.exe http 8081

  • si no estamos registrados dará error
