Testing Unitarios con jest

  1. Al crear el proyecto se marca la casilla Unit Testing
    además de Babel, router, y typeScript etc
  2. Todo lo demás será por defecto, al llegar a testing hay 2 opciones en este caso se elegirá jest
  3. El archivo de test se ubica dentro de: test/unit
  4. la extensión es: nombreDeArchivo.spec.ts
  5. en este caso se comprobará: que pinta la acción y que increment o decrement
