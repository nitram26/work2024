V-MODEL 1228

  1. Crear el input v-text-field, con el v-model; la variable es relativa
  2. otro input tipo v-checkbox, con v-model y su variable
  3. 2 h3 con if/show dando las clases y si acepta o no
  4. la igualdad es con === 
          

            <template>
            <h1>V-MODEL </h1> 
           <v-row class="pt-6">
            <v-col cols="3" >
            <v-text-field
              v-model="texto"
              label="Escribir aquí"
              outlined
          />
          Acepta términos y condiciones
          <v-checkbox
          class="pl-12"
          v-model="check"
          />
          </v-col>
          <v-col cols="3">
            <h3>{{ texto }}</h3><br><br><br>
          
            <h3 v-show="check===true" class="text-green">Acepta Términos y condiciones</h3>
            <h3 v-show="check===false" class="text-red">No Acepta Términos y condiciones</h3>
          </v-col>
          </v-row>
          </template>
          <!-- ******************************* --> 
          <script setup>
          import { ref } from 'vue';
          
          
          let texto = ref('')
          let check = ref(true)
          </script>
          <!-- ******************************* -->
          <style scoped>
          div{
            background-color: antiquewhite;
           
          }
          
          </style>
