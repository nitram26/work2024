LIVE SERVER

PROBLEMA CON EL PUERTO

    SOLUCIÓN 1
  1. el puerto necesita a veces dos lugares para ser configurado, en el setting.json
  2. pero a veces se a modificado por algún tipo de error o conflicto
  3. se va al administrador se pone live server, se busca el puerto y se va a el area de trabajo, donde también es preciso modificarlo
    4. SOLUCIÓN 2
  4. A veces, Live Server guarda configuraciones en caché. Para resolverlo:
  5. • Elimina la carpeta .vscode del proyecto si existe (esto eliminará configuraciones específicas del proyecto, asegúrate de revisarlas antes).
  6. • Reinicia el servidor nuevamente.
