LIVE SERVER

PROBLEMA CON EL PUERTO

    SOLUCIÓN 1
  1. Abrir manualmente un html para ver en que puerto abre
  2. Revisar el puerto que le hemos asignado en la app
  3. Revisar el vscode del propio proyecto donde hay un setting.json también y cambiar el puerto

  4. En Live Server el puerto necesita a veces dos lugares para ser configurado, en el setting.json y se entra a config se pone Live Server Config y aparece otro lugar
  5. A veces se a modificado por algún tipo de error o conflicto

  6. Live Server guarda configuraciones en caché. Para resolverlo:
  7. • Cambia o Elimina la carpeta .vscode del proyecto si existe (esto eliminará configuraciones específicas del proyecto, asegúrate de revisarlas antes).
  8. • Reinicia el servidor nuevamente, cerrando la app y abriéndola
