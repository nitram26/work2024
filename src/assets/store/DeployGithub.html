DESPLEGAR UN PROYECTO EN GITHUB 

    

      Para desplegar un proyecto HTML y CSS en GitHub Pages, sigue estos pasos:

      ### 1. Crea un Repositorio en GitHub
      1. Ve a [GitHub](https://github.com/) y asegúrate de estar registrado.
      2. Haz clic en **New Repository** (Nuevo Repositorio).
      3. Dale un nombre al repositorio (por ejemplo, `mi-proyecto`), selecciona la opción **Public** (Público) 
 y luego haz clic en **Create repository** (Crear repositorio).
      
      ### 2. Sube los Archivos del Proyecto
      1. En el repositorio recién creado, verás opciones para **Add file** 
(Agregar archivo). Puedes elegir **Upload files** (Subir archivos) para cargar los archivos HTML, CSS y 
 cualquier otro archivo relacionado directamente desde tu computadora.
      2. Arrastra y suelta los archivos del proyecto o selecciona cada archivo manualmente.
      3. Una vez subidos, escribe un mensaje de confirmación (commit message) y haz clic en **Commit changes** (Confirmar cambios).
      
      ### 3. Activa GitHub Pages
      1. Ve a la sección **Settings** (Configuración) de tu repositorio.
      2. Desplázate hacia abajo hasta la sección **Pages** en el menú de la izquierda.
      3. En la opción **Source** (Fuente), selecciona la rama `main` o `master`, 
dependiendo de cómo esté configurado tu repositorio, y selecciona la carpeta **root** (raíz).
      4. Haz clic en **Save** (Guardar).
       
      ### 4. Accede a tu Sitio
      1. GitHub tardará unos segundos en desplegar tu proyecto. Verás una notificación con el enlace de tu sitio.
      2. El enlace será algo como `https://nitram26.github.io/vitalis`.
      
      ¡Listo! Ahora tu proyecto HTML y CSS está en línea usando GitHub Pages.
