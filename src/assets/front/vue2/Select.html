      SELECT
      <v-select
        :items="boxsmall"
        item-text="correlative"
        item-value="correlative"
        label="Selecione un Vale"
        v-model="selectedItem"
      />
      
      •  :items="boxsmall":, boxsmall debe ser una matriz (array) en el componente Vue que contiene los elementos a mostrar en el dropdown. 
      •  item-text="correlative": Este atributo indica qué propiedad de cada objeto en la matriz boxsmall se usará para mostrar el texto de cada opción en el dropdown. 
      •  item-value="correlative": Similar a item-text, este atributo indica qué propiedad de cada objeto en la matriz boxsmall se usará como el valor de cada opción en el dropdown
      •  v-model="selectedItem": El atributo v-model enlaza el valor seleccionado del dropdown con la propiedad selectedItem del objeto data de Vue. Esto significa que cuando el usuario selecciona una opción del dropdown, el valor de selectedItem se actualiza automáticamente para reflejar la selección actual.
