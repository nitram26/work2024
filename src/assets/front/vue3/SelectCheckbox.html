CHECKBOX 0705

  1. Realizado con vuetify
  2. En html seria diferente la sintaxis
  3. Que muestre si acepta usando v-show y color azul
  4. por defecto acepta, si cambia muestre no acepta y cambia de color

CON VUETIFY

          

            <template>
            <div class="work">
             <h1>Practica Julio 2024</h1> 
             <v-checkbox v-show="acepto ===true"  v-model="acepto" class="azul">acepto  </v-checkbox>
             <v-checkbox v-show="acepto ===false"  v-model="acepto" class="rojo">no acepto  </v-checkbox>
            </div>
            
          </template>
          <!-- ******************************* --> 
          <script setup>
          import { ref } from 'vue';
          
          let acepto = ref(true)
          
          </script>
          <!-- ******************************* -->
          <style scoped>
          .work{
            background-color: burlywood;
            height: 5em;
            margin-left: 3em;
          }
          .rojo{
            color: red;
          }
          .azul {
            color: blue;
          }
          </style>
          <!-- ******************************* -->
