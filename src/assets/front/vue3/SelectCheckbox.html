SELECT & CHECKBOX 0805

  1. Realizado con vuetify
  2. En html seria diferente la sintaxis
  3. PARA EL CHECKBOX
  4. Que muestre si acepta usando v-show
  5. por defecto acepta, si cambia muestre no acepta
  6. PARA EL SELECT
  7. Usar v-select
  8. los elementos van en un array en el script
  9. :items="items"
    para traer el array u objeto
  10. v-model = "item1"
    va vacío para que capture el elemento seleccionado y lo muestre
