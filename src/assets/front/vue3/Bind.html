    Tarea

  1. Hacer un h3 con el texto: Esta es una etiqueta que cambia de color
  2. color por defecto azul
  3. hacer un botón que al pulsar cambie el color del texto
  4. cambia a rojo
  5. despues de 2 segundos regrese a su color

    6. Información

  6. v-bind:  : SIGNIFICA UNIR, CONECTAR, ASOCIAR
  7. Permite modificar el atributo que se pone después del bind
  8. Es unidireccional
  9. Se usa para clases y estilos
  10. Se pueden aplicar a una etiqueta varias clases en un solo bind
  11. Trabaja con variables, por eso se traen del SCRIPT
  12. v-bind relaciona datos del STYLE on el template, es unidireccional, se encierra entre llaves 
    

      <template>
  <div>
    <h1>Bind</h1>
    <h1 :class="color">esta es una etiqueta que cambiará de color</h1>
    <button class="btn btn-primary" @click="cambio">CAMBIA</button>
  </div>
</template>
<!-- ************************************* -->
<script setup>
import { ref } from 'vue';

let color = ref('uno');

let cambio = () => {
  color.value = 'dos';
  setTimeout(() => {
    color.value = 'uno';
  }, 3000);
}

</script>
<!-- ************************************* -->
<style scoped>
.uno {
  background-color: antiquewhite;
  color: brown;
}
.dos {
  background-color: brown;
  color: aliceblue;
}
</style>
