USO DEL V-BIND => : 0435

  1. v-bind:  : SIGNIFICA UNIR, CONECTAR, ASOCIAR
  2. Permite modificar el atributo que se pone después del bind
  3. Es unidireccional
  4. Se usa para clases y estilos
  5. Se pueden aplicar a una etiqueta varias clases en un solo bind
  6. Trabaja con variables, por eso se traen del SCRIPT
  7. v-bind relaciona datos del STYLE on el template, es unidireccional, se encierra entre llaves 
    

    
    
    
Poner un  párrafo que cambie de color con un botón 

    
    <template>
      <div>
       <h1 class="mb-12">VUE3</h1> 
        <h2 :class="color">Este es una etiqueta que cambia de color </h2>
    
        <v-btn @click="cambio">PULSAR</v-btn>
       <br><br><br>
      
      </div>
    </template>
    <!-- ******************************* --> 
    <script setup>
    import { ref } from 'vue';
    
    let color = ref('red')
    let cambio= ()=>{
      color.value= 'danger' 
    }
    
    </script>
    <!-- ******************************* -->
    <style scoped>
    .danger{
      color: red;
    }
    .red {
      color: blue;
    }
    </style>
