INJECTION 0651

  1. Padre: Se importa el hijo, se crea la variable y se envía con provide
  2. Primero se pone el nombre de envio entre comillas simples
  3. y a continuación la variable que se envia, sin comillas
  4. Ejemplo: provide('envio', num)
  5. Hijo: se recibe con el inject
  6. Ejemplo : let producto = inject('envio')
  7. Se muestra la variable producto
  8. Para cambiar se crea una función en el hijo con un botón
  9. Y el valor que se quiere cambiar. Listo
