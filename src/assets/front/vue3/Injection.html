INJECTION

  1. Se usa para traer elementos de una página a otra
  2. EL Padre envia los datos usa el provide
  3. el hijo los recibe con inject
  5. Padre: Se importa el hijo, se crea la variable y se envía con provide
  6. Primero se pone el nombre de envio entre comillas simples
  7. y a continuación la variable que se envia, sin comillas
  8. Ejemplo: provide('envio', num)
  9. Hijo: se recibe con el inject
  10. Ejemplo : let producto = inject('envio')
  11. Se muestra la variable producto
  12. Para cambiar se crea una función en el hijo con un botón
  13. Y el valor que se quiere cambiar. Listo
