  3. BINDEO BIDIRECCIONAL v-model

    Es reactivo se usa con ref
    El v model relaciona un elemento del template con otro del script 
    

      <template>
      <div>
          <h1>VUE3</h1>
          <v-row>
            <v-col cols="4">
            <v-form>
                <v-text-field
                placeholder="Escribe aquí"
                density="compact"
                v-model="input"
                ></v-text-field>
              </v-form>
            </v-col>
          </v-row>
          <h3>{{ input }}</h3>
      </div>
    </template>
    <!-- ************************************* -->
    <script setup>
    import { ref } from 'vue';
    
    let input = ref('')
    </script>
