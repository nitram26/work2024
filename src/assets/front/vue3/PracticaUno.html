Objetivo

  1. mostrar saldo
  2. Botones de depósito y retiro
  3. si esta en o que no pueda retirar //usar un alert
  4. cantidad de movimientos
  5. listado de movimientos
  6. Dar estilos

Proceso

  1. Hacer el let referencial y llamarlo en el template
  2. Crear un botón con la función mas
  3. Hacer la lógica de aumentar
  4. Duplicar y cambiar a disminuir
  5. Crear el boton menos
  6. No puede haber saldo negativo
  7. crear una condicional que emita un alert
  8. H3 la cantidad de movimientos en let ref
  9. Mostrar si ha sido un deposito o retiro de 100
  10. Definir un array ref
  11. Aumentar la lógica con un push
  12. Llamarlo con un v-for
  13. Aunmentar la logica para disminuir
  14. Darle estilo
          

            <template>
            <div>
             <h1>Practica Uno</h1>  <br><br>
              <h2>Ahorros  $.{{ ahorro }}</h2> <br><br>
              <h2>Cantidad de movimientos {{ movimiento }}</h2> <br><br>
              <v-btn @click="deposito" class="btn">Depósito</v-btn>
              <v-btn @click="retiro">Retiro</v-btn> <br><br><br>
           
              <h3>MOVIMIENTOS </h3>
              <ol>
                <li v-for="(mov, index) in movimientos" :key="index">{{ mov }}</li>
              </ol>
            </div>
          </template>
          <!-- ******************************* --> 
          <script setup>
          import { ref } from 'vue';
          
          
          let ahorro = ref(0)
          let movimiento = ref(0)
          let movimientos = ref([])
          
          let retiro = ()=>{
            if (ahorro.value > 0) {
              ahorro.value -= 100
              movimiento.value++
              movimientos.value.push('Retiro $100.00')
              
            }else{
              alert('No tiene Fondos')
            }
          }
          let deposito = ()=>{
            ahorro.value += 100
            movimiento.value++
            movimientos.value.push('Deposito $100.00')
          }
          </script>
          <!-- ******************************* -->
          <style scoped>
          div{
            background-color: blanchedalmond;
            margin: 4em;
            padding: 4em;
          }
          .btn{
            margin-right: 2em;
          }
          h2{
            margin-left: 3rem;
          }
          </style>
