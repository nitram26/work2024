LLAMAR API PLACEHOLDER

  1. hacer el componente
  2. llamarlo en el View
  3. hacer la tabla en el componente, si se usa vuetify: v-data-table
  4. se llama los datos usando :items="datum"
  5. se crea un array datum ref vacío
  6. con esto enlazamos el array al template

    7. LLAMAR LA API

  7. Se llama a trabes de un onMounted()
  8. dentro un async como función flecha
  9. dentro de la función un try-catch
  10. dentro del try entre llaves un let response = await axios.get ('ruta de la api') en este caso:'https://jsonplaceholder.typicode.com/users'

    11. Instalar axios

  11. Para llamar axios debe de estar instalado
  12. npm install axios
  13. pero puede suceder que no deje instalarlo,
  14. se pude deber a que no es compatible con la versión (en mi caso de typescript)
  15. se actualiza la versión con el comando
  16. npm install typescript@latest

    17. Capturar los datos

  17. se asigna a datum.value el response.data.map({})
  18. dentro de las llaves un objeto con los nombres de cabecera y el llamado que se hace
  19. ejemplo NOMBRE : user.name,
  20. el catch (error )
  21. entre llaves un console.error
  22. y dentro 'revisar', error
  23. NOTA
  24. también es posible llamar a los datos usando un v-for
    25. v-for="(nombre, index) in datum" :key="index">{{ nombre.NOMBRE }} 
          
Componente Hijo

     

      <template>
  <v-data-table
    :items="datum"
  >

  </v-data-table>
</template>
<!-- ******************************* --> 
<script setup>
import { onMounted, ref } from 'vue';
import axios from 'axios';
let datum = ref([])
onMounted ( async () => {
  try {
    const response = await axios.get ('https://jsonplaceholder.typicode.com/users')
    datum.value = response.data.map(user => ({
      Nombre :user.name,
       Alias : user.username,
       Correo : user.email
    }))
  } catch (error) {
    console.error('revisar', error)
  }
})


</script>
