PWA en vue3

PWA
  1. Crear un proyecto nuevo
  2. Generar el proyecto manualmente
  3. incluir PWA
  4. Todo lo demás por defecto
  5. dirigir se al directorio de trabajo si no se a limpiado GIT
  6. ejecutar el constructor con: npm run build
  7. instalar el servidor de prueba : npm install http-server -g
  8. Borrar el caché de chrome para evitar conflictos (ver store/windows/borrar datos de caché)
  9. levantar el dist con http-server dist/
  10. el servicio se levanta para iniciar la configuración del manifest.json y Service Worker
  11. Se debe a que un service worker solo funciona a través de un https, un navegador seguro

    12. TUNELADO

  12. Se hace con ngrok: https://download.ngrok.com/windows
  13. Descargar el ejecutable e instalarlo
  14. Mostrará una consola donde ejecutaremos: ngrok.exe http 8080

    15. Registro

  15. si no estamos registrados dará error
  16. Si no tienes una cuenta, ve a https://dashboard.ngrok.com/signup y regístrate.
  17. en la terminal aparece https://dashboard.ngrok.com/signup y nos dará un token
  18. ngrok config add-authtoken 2qOfOrs9pEYlOVwrvJzc2gohs2c_7xtxumV8LDcRQQJKLFpQv
  19. se ejecuta en la terminal y guarda el acceso
  20. Authtoken guardado en el archivo de configuración: C:\Users\cm7ch\AppData\Local/ngrok/ngrok.yml
  21. Se levanta el servicio con el link que proporciona ngrok
  22. Y debera aparecer el icono de instalar app
  23. La app estará instalada en la PC y en el buscador se pone pwa y aparece la app
  24. El manifest es para configurar la app
  25. El service worker para optimizar el rendimiento

Elaboración del MANIFIESTO

  1. Se busca el manifest.json
  2. todo aparece desordenado y en una linea; Se da formato con ALT+SHIFT+F
  3. El manifest.json se mueve a public y se elimina de la carpeta dist
  4. el name es el nombre que se muestra en la app
