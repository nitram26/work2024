VUE-VITE

  1. tener instalado node
  2. Proceso
  3. npm init vue@latest
  4. cd carpeta
  5. npm install
  6. add vuetify
  7. npm run dev
  8. Proyecto terminado

VUE-CLI

  1. tener instalado node
  2. Tener instalado el CLI de Vue asi:
  3. npm install -g @vue/cli
  4. Proceso
  5. vue create nombreDelProyecto
  6. elegir manual o vue3
  7. npm add vuetify
  8. Proyecto terminado
  9. npm run serve
  10. Bootstrap necesita 2 instalaciones:
  11. npm install --save bootstrap
  12. npm install --save @popperjs/core
