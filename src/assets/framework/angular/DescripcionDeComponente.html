DESCRIPCIÓN DEL COMPONENTE

  1. DENTRO de un componente Angular crea una serie de archivos, vemos 6.
  2. APP.COMPONENT.CSS, es el STYLE
  3. LA MANERA GLOBAL ES USAR EL STYLE
  4. APP.COMPONENT.HTML, es lo que se va a mostrar
  5. APP.COMPONENT. SPEC.TS, ES para realizar pruebas
  6. APP.COMPONENT.TS, este es el SCRIPT de nuestro componente
  7. COMPONENT
  8. SELECTOR, es la ruta de llamado del componente
  9. TEMPLATEURL, llama al documento HTML que se va a mostrar
  10. STYLEURL, los estilos para ese documento HTML
