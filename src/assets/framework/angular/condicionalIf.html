CONDICIONAL IF

  1. Un input que muestre por consola lo escrito
  2. Esto a traves de un botón
  3. Una condicional que muestre SE HA ENVIADO MENSAJE al pulsar el botón
  4. Un botón cerrar
  5. Y un else que muestre Mensaje de prueba mientras no se envié
    6. NOTA IMPORTANTE
  6. Para usar las condicionales se tiene que importar el CommonModule
  7. Tanto en el appModules.ts como en el ts del componente
    

    
AGREGAR EL COMMON MODULE EN  APPMODULES.TS

    import { NgModule } from '@angular/core';
    import { BrowserModule } from '@angular/platform-browser';
    import { CommonModule } from '@angular/common';
    
    @NgModule({
      declarations: [
       
       
      ],
      imports: [
        BrowserModule,
        CommonModule
      ],
      providers: [],
     
    })
    export class AppModule { }
    
    
EN HTML


    <input 
    type="text"  
    placeholder="Ingresar Nombre" 
    class="form-control m-2" #name>
    
    <button class="btn btn-primary m-2" (click)="mostrarConsola(name.value)">Mostrar en Consola</button>
    
    <div class="alert alert-primary mt-4" *ngIf="msg; else mensaje">
      <p>SE HA ENVIADO EL MENSAJE</p>
      <button (click)="close()" class="btn btn-success">CERRAR</button>
    </div>
    
    <ng-template #mensaje>
      <div class="alert alert-dangerFOR">mensaje de prueba </div>
    </ng-template>
