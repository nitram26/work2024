Binding con Clases

  1. Tres botones cambien el color del texto
  2. luego un botón que habilite y deshabilite los botones
  3. las clases para css se agregan solo con la palabra class
  4. El click se agrega entre parentesis
  5. las variables se ponen [class] = "nombreDeVariable"
          

            HTML
           
            <button [class.disabled]="buttonDisabled" class="btn btn-primary"(click)="color='text-primary'" >Color 1</button> 
            <button [class.disabled]="buttonDisabled" class="btn btn-success" (click)="color='text-success'" >Color 2</button> 
             <button 
             [class]= "{
              disabled :buttonDisabled}" 
              class="btn btn-danger" 
              (click)="color='text-danger'" 
              >
              Color 3</button>
            
            
            <h4 [class]="color">Texto que cambiará de color </h4>
            
            <br><br><br><br><br><br>
            <button (click)="buttonDisabled = !buttonDisabled " class="btn btn-success">
              habilitar / Deshabilitar 
            </button>
