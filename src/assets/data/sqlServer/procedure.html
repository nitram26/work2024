PROCEDIMIENTO ALMACENADO

  1. Son consultas que se almacenan en la misma base de datos
  2. Su objetivo es simplificar la consulta usando variables
    

      
CONSULTA SQL

      SELECT idBase, fechaCreacion
FROM Cliente
WHERE YEAR(fechaCreacion) = 2023
  AND MONTH(fechaCreacion) = 7
  AND idBase = 1

  
CREACIÓN DE PROCEDIMIENTO

      CREATE PROCEDURE sp_BuscarPorFechaYBase
      @year INT,
      @month INT,
      @idBase INT
  AS
  BEGIN
      SELECT idBase, fechaCreacion
      FROM Cliente
      WHERE YEAR(fechaCreacion) = @year
        AND MONTH(fechaCreacion) = @month
        AND idBase = @idBase;
  END;
  
  
Consulta

  EXEC sp_BuscarPorFechaYBase @year = 2024, @month = 7, @idBase = 1;
